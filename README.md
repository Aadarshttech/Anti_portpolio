# Anti Portfolio

Welcome to the Anti Portfolio project! This is a modern portfolio website built with Next.js and Tailwind CSS.

## Project Structure

- **portfolio/**: Contains the main Next.js application source code.
- **Nepalcricket_MLprediction-main/**: Contains machine learning prediction scripts/data for Nepal Cricket.

## Getting Started

1.  Navigate to the portfolio directory:
    ```bash
    cd portfolio
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
