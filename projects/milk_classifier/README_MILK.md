# 🥛 Milk Quality Classifier Setup Guide

To run this application on another computer, follow these simple steps:

### 1. 📂 Required Files
Copy these two files to the new computer:
- `milk_classifier.py`
- `milk_model.pkl`

### 2. ⚙️ Install Python Libraries
Open a terminal (Command Prompt or PowerShell) and run:
```powershell
pip install pandas numpy scikit-learn joblib
```

### 3. 🚀 Run the App
Navigate to the folder and run:
```powershell
python milk_classifier.py
```

---
*Note: If using Linux, you may also need: `sudo apt-get install python3-tk`*
